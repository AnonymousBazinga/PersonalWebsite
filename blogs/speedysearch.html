A super speedy search engine with LLMs

Try it at fast.browse.dev. A peek into the future of search engines, built with ChatJimmy and Exa.

The code is open-sourced here. Eventually I'd like to do a breakdown of how custom chips optimize speed.

Claim: LLM speed is a huge bottleneck for a lot of valuable applications. I think the extent of this is wildly underestimated. Currently ChatJimmy only serve Llama-3.1 8B, which is smart enough for a simplistic search engine. I am excited for the future where more intelligent models will be served at this speed enabling more complex agents to be near-instant. I believe people are strongly understating the impact that we could have if every model was being served at this speed.
